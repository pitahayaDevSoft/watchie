# Soul & Identity

## The Project's Soul

We seek to provide an elegant, terminal-native gateway to cinema, freeing the user from heavy web layouts and trackers, while allowing direct integration with local media players.

## Guiding Principles

1. **Terminal Sovereignty**: Key-driven navigation, fast startup, and native image rendering (Kitty) are first-class citizens.
2. **Minimalist Integration**: Unifying TMDB and playimdb.com with minimal dependencies and robust fallback strategies.
3. **Rust Integrity**: Zero-warning build safety, clean async task scheduling, and error boundaries.
