# Development Guide

## Prerequisites

- Rust toolchain (stable, edition 2021)
- Cargo

## Local Setup

1. Clone the repository.
2. Build the project:
   ```bash
   cargo build
   ```
3. Run the binary:
   ```bash
   cargo run
   ```

## Useful Commands

- `cargo run`: Run the TUI client.
- `cargo run -- search "Inception"`: Search via CLI.
- `cargo check`: Check compilation.
- `cargo test`: Run tests.
